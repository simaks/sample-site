# Sample site

Live site: [https://simaks.github.io/sample-site/](https://simaks.github.io/sample-site/)

## Installation
```bash
npm install
```

## Http server
```bash
npm run serve
```

## Sass watcher
```bash
npm run watch:sass
```

## Development
This will start http-server and sass watcher
```bash
npm run dev
```
